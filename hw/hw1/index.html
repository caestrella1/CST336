Overview

Invented by Gottfried Leibniz, binary is a base-2 number system composed of zeroes and ones. In this lesson, you will learn why computers use binary, how to convert between binary and decimal (base-10), and how to perform basic arithmetic.

Section Description
What is it? Here we will discuss what the binary system is, how it works and why computers use it.
Conversion After learning what binary is, we will cover how to read it and convert to and from decimal (base-10).
Arithmetic Finally, we will learn how to perform basic arithmetic, such as addition and subtraction, on binary numbers.
